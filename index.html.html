Passo a Passo para o Processo de Compras

  1. Solicitação de Compra: Acesse o menu "Suprimentos -> Compras -> Solicitações de Compra -> Gerar SC" para criar uma nova solicitação de compra.
  2. Preencher informações: Complete todos os campos obrigatórios com informações detalhadas sobre o item ou serviço necessário.
  3. Submeter para aprovação: Após preencher todas as informações, submeta a solicitação para o fluxo de aprovação.
  4. Aprovação: Os aprovadores devem acessar "Suprimentos -> Compras -> Solicitações de Compra -> Autorizar SC" para analisar e aprovar/reprovar a solicitação.
  5. Criação do PAC: Após aprovada, acesse "Suprimentos -> Compras -> PAC - Processo Compras -> Gerar PAC" para dar início ao processo administrativo de compras.
  6. Documento de Cotação: Para solicitar cotações de fornecedores, acesse "Suprimentos -> Compras -> PAC - Processo Compras -> Cotar Preço de Materiais" para gerar o documento PDF.
  7. Acompanhamento: Acompanhe o status do processo através dos menus correspondentes em cada etapa.
